# LumiTask 产品定位（终版）

## 核心定位

> **LumiTask = OpenClaw 的任务管理面板**
> 只管理用户**明确要求记录**的任务，不自动收集。
> OpenClaw 是引擎，LumiTask 是仪表盘。

## 什么进 LumiTask

| 进 LumiTask | 不进 LumiTask |
|-------------|---------------|
| 用户说"记到任务/创建任务/添加待办" | 即时对话（翻译、问答、改文案） |
| 用户说"每天帮我做X"（定时任务） | 长任务（除非用户明确说记录） |
| 用户说"记一下这个想法"（收集箱） | 一次性聊天完成的工作 |
| 用户在 Web 面板手动创建 | |

**原则：不主动记录，只在用户明确要求时才写入。**

## 信息架构

```
侧边栏：
  📊 Dashboard         — 概览（统计 + 最近执行 + 定时任务日程）
  📋 Tasks             — 看板 + 日历视图（可切换）
  📥 Inbox (3)         — 收集箱（显示未处理数量）
  🔄 定时任务           — 读取管理 OpenClaw Cron
  🤖 Agents
  ⚙️ Settings
```

### 📥 Inbox（收集箱）

**核心逻辑**：
- 跟任务一样关联 agent（默认 agent 或用户选择）
- 有提醒机制：默认 3 天汇总提醒一次
- 通过 OpenClaw agent 给用户发消息

**数据模型**：复用 tasks 表，`status = 'inbox'`

**提醒机制**：
```
Settings 配置：
  收集箱提醒: [✓] 开启
  提醒频率: 每 [3] 天
  提醒 Agent: [🐶 旺财 ▼]

提醒内容（通过 openclaw cron 发送）：
  "📥 收集箱有 3 条待处理：
   1. 下周整理季度报告 (5天前)
   2. 研究 Rust async 生态 (3天前)
   3. 给 openclaw 提 PR (1天前)
   是否需要安排执行？查看: http://localhost:3000/inbox"
```

**初始化引导**：首次打开 LumiTask 时弹出配置向导：
```
┌──────────────────────────────────┐

│  [🐶 旺财 (OpenClaw) ▼]          │
│                                   │

│  [✓] 定期提醒处理收集箱            │
│  每 [3] 天通过 Agent 发送汇总     │

```

**UI**：

操作：
- **快速添加**：输入标题 + 选 agent → 进入 inbox
- **创建任务并执行**：转为 `immediate` 任务，立即执行
- **转为任务**：转为 `manual` 任务，进入看板 Open 列
- **删除**：标记为 `cancelled`

当用户在 OpenClaw 聊天中处理了某个收集箱项目：
- 旺财执行后调 `lumitask complete <编号>` → 状态从 inbox 变为 done
- LumiTask 界面实时同步

### 🔄 定时任务（OpenClaw Cron）

读取 + 管理 OpenClaw 的 cron 任务：

```
┌──────────────────────────────────────────────────┐
│  定时任务 (2)                     [+ 新建]        │
│                                                   │
│  ┌────────────────────────────────────────────┐  │
│  │ 🟢 每日 GitHub PR 检查                     │  │
│  │    Agent: 🐶 旺财                          │  │
│  │    Cron: 0 9 * * * (每天 09:00)            │  │
│  │    下次: 明天 09:00                         │  │
│  │    上次: 今天 09:00 ✅ 成功                 │  │
│  │    [编辑] [暂停] [立即执行] [▼ 历史]        │  │
│  └────────────────────────────────────────────┘  │
│                                                   │
│  ┌────────────────────────────────────────────┐  │
│  │ ⏸ 每周工作总结 (已暂停)                    │  │
│  │    Agent: 🐶 旺财                          │  │
│  │    Cron: 0 10 * * 1 (每周一 10:00)         │  │
│  │    [编辑] [启用] [▼ 历史]                   │  │
│  └────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────┘
```

后端通过 `openclaw cron` CLI 操作：
- 列表：`openclaw cron list --json`
- 新建：`openclaw cron add --cron "..." --agent main --message "..."`
- 编辑：`openclaw cron edit <id> ...`
- 暂停/启用：`openclaw cron disable/enable <id>`
- 执行：`openclaw cron run <id>`
- 历史：`openclaw cron runs --json --id <id>`

## Settings 增加

```
Settings:
  默认 Agent: [🐶 旺财 ▼]
  默认工作目录: [~/Downloads] 📂

  收集箱提醒:
    [✓] 开启
    频率: 每 [3] 天
    提醒 Agent: [🐶 旺财 ▼]

  通知:
    [✓] 应用内通知
    [✓] 浏览器通知
```

提醒实现：Settings 保存后，调 `openclaw cron add` 创建一个提醒 cron job：
```bash
openclaw cron add \
  --cron "0 9 */3 * *" \
  --agent main \
  --announce \
  --message "请检查 LumiTask 收集箱：运行 lumitask list --status inbox 查看待处理项目，汇总后告诉用户。"
```
